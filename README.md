# Название проекта: BurgerPlusManagementSystem
Данная система предназначена для управления пользователями

#Реализованный функционал:
#Авторизация с капчей
#Окно пользователя
#Окно администратора с возможностью добавления новых пользователей, редактирования уже существующих и разблокировкой заблокированных пользователей.

#Используемые технологии и программные средства:
#IDE Visual Studio 2022
#SQL Server Managment Studio 2019
#Язык программирования C#
#Фреймворк для связи с БД Entity Framework 6.2.0

#Требования к ПО:
#ОС: Windows 10/11
#ЦП: Intel Core 2 duo или выше
#RAM: от 4гб
#Видеокарта: geforce gtx 1050

#Порядок установки:
1. Необходимо установить SQL Server Managment Studio
2. Скачать DataBase.sql и создать из него БД
3. Установить программу
4. В файле app.config указать адрес вашего sql сервера
